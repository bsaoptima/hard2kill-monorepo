# Hard2Kill

Skill-based PvP betting platform. Monorepo containing all code and strategy.

## Subprojects

| Folder | What |
|--------|------|
| `hard2kill/` | Main platform — Next.js web, Colyseus realtime, Supabase, Stripe |
| `geostakes/` | GeoGuessr-style 1v1 wager MVP (Next.js + Supabase Realtime) |
| `cs16-wagering/` | CS 1.6 WASM build + wagering integration |
| `cs/` | CS droplet infrastructure and AMX config |
| `coworker/` | Strategy docs: vision, product, marketing, engineering, knowledge |

## Getting started

Each subproject has its own `package.json` and run instructions. See per-folder READMEs.

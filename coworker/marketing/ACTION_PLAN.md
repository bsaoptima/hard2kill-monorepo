# The Plan — 30 Days to First Real Players

## Week 1: Fix the product (so it doesn't embarrass you)

**Day 1-2: Code changes (a few hours)**
- Remove 3-game bot cash limit → players can keep playing
- Auto-grant 50 coins on signup → nobody starts at zero
- Replace fake "4 players online" with nothing, or "X matches played today"

**Day 3-5: Build challenge links**
- `hard2kill.gg/c/[id]` → "Stefan challenged you to a $5 match"
- Clicking it → signup → match starts between both players
- Shareable to WhatsApp, iMessage, Twitter, anywhere
- This is your ONLY growth mechanic that matters right now

**Day 6-7: First deposit bonus**
- Deposit $5, get $5 free
- Show it prominently on landing page

Nothing else. Don't touch anything else in week 1.

---

## Week 2: Get 20 players by hand

**The only channel that works at zero scale: DM people individually.**

**Where to find them: wager Discord servers.**

There are thousands of people who already bet money on 1v1 matches using manual middlemen in Discord. They deal with scams, slow payouts, middlemen going offline. You solve all of that.

**Day 8-9:**
- Join 15 wager/competitive Discord servers (DISBOARD → search "wagers", "1v1")
- Lurk. Read the chat. Understand how they talk, what games they play, what annoys them.

**Day 10-14: DM 50 people**

Not a server post. Not a link drop. Individual DMs to active members.

```
Hey — saw you do wager matches here. I built a platform that does 
automated escrow for 1v1 money matches (no middleman needed). 

I'll put $10 in your account free if you want to try it. 
No catch, just need early players to test it.

[link]
```

50 DMs → 10-15 will respond → 5-10 will actually try it.

Also: DM 5 middlemen specifically. They're doing manual work that your platform automates. Pitch them as affiliates.

---

## Week 3: Create content from what happened

By now you have 10-20 accounts with some match history.

**Day 15-17:**
- Record yourself playing 5-10 matches (vs bots or real players)
- Edit into clips using CapCut:
  - "I put $10 on myself in a 1v1. Here's what happened." (60s)
  - "This match came down to the last second." (30s)  
  - "Turned $5 into $25 in one session." (45s)
- Post across TikTok + YouTube Shorts + Instagram Reels. 3/day.

**Day 18-21:**
- Keep posting 3 clips/day
- Send challenge links to 10 people you actually know (friends, coworkers, anyone)
- Give each $10 free
- Stream one 1-hour session on Kick: "1v1 me for money"

---

## Week 4: Double down on what worked

**Day 22-28:**
- Look at the numbers. Where did signups come from?
- If Discord DMs worked → do 50 more
- If a TikTok clip got views → make 5 more like it
- If challenge links got shared → make sharing easier (post-win "challenge a friend" prompt)
- Host one $25 freeroll tournament (free entry, cash prize) and announce it in every Discord server you joined. Free tournaments are not self-promo — they're events. Mods allow these.

## What NOT to do

- Don't post on Reddit
- Don't run ads
- Don't build a ranking system, tournaments, cosmetics, or new games
- Don't optimize SEO
- Don't contact press
- Don't build an affiliate dashboard
- Don't do anything that isn't: fix product → DM people → make content

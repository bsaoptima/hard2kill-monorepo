# Hard2Kill Coworker Space

This is the strategic collaboration hub for Hard2Kill. The actual codebase lives at `/Users/stefan/Desktop/Code/game/hard2kill`.

## Structure
- `vision/` — Company direction, market thesis, competitive landscape
- `product/` — Roadmap, feature specs, game design, UX decisions
- `marketing/` — Growth strategy, messaging, launch plans, channels
- `engineering/` — Architecture decisions, scaling strategy, tech debt
- `knowledge/` — Research, learnings, accumulated decisions

## How to work here
- When making strategic decisions, document the reasoning in the relevant folder
- Cross-reference the actual codebase when needed for technical context
- Build on existing documents rather than creating duplicates
